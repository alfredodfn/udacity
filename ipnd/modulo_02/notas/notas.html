A World Wide Web

A WWW (World Wide Web) é uma coleção de documentos HTML (HyperText Markup Language), o principal tipo de documento que nela pode ser contido. Entretanto, diversos outros tipos de arquivos são suportados: imagens, vídeos, músicas, textos planos, entre outros. A comunicação entre esses documentos é realizada por meio de Hyperlinks.

Componentes da Web

A Web é composta pelo cliente (você, seu computador e um navegador), a Internet e os servidores que armazenam os documentos HTML. O cliente, através do navegador, interage com o servidor por meio do protocolo HTTP (Hypertext Transfer Protocol), requisitando os documentos HTML e demais tipos associados.

Visão geral sobre HTML

Os elementos HTML são utilizados para indicar como o conteúdo de deve ser exibido na tela do navegador. Seu formato consiste em uma tag de abertura e uma de fechamento com algum conteúdo entre elas. Existe a possibilidade de algumas tags não possuírem conteúdo, caso em que são denominadas de tags vazias (void). Os elementos podem ser blocos, quando uma caixa imaginária é criada no entorno para permitir formatações mais complexas, ou inline, para formatações mais simples e diretas.

Os elementos aceitam atributos que podem ser colocados nas tags de abertura. Os atributos dão informações adicionais para permitir comportamentos diferentes a um elemento em específico. Um exemplo é o atributo href para a tag a, que permite informar o link a ser inserido.

Os principais elementos aprendidos até agora são:

  • p
  • a
  • b
  • div
  • h1
  • em
  • spam

Estrutura dos Documentos

Até este ponto trabalhamos com duas linguagens principais: HTML, responsável pela estrutura da página web, e CSS, responsável pelo estilo, pela aparência que as páginas possuem.

A estrutura das páginas é representada na forma de árvore, onde cada elemento ou conteúdo de um elemento é um nó, e denominada Document Object Model ou DOM. Ou seja, DOM nada mais é que a estrutura de árvore da página.

Embora existam diversos elementos de variadas formas, todos os elementos estão contidos em blocos retangulares (ou caixas), que facilitam a disposição dos elementos em tela. A divisão em caixas permite segmentar conteúdos e generalizar estilos e formatações, tornando o código escrito menos repetitivo, mais bonito e principalmente mais legível.
